# Lightroom Presets - High-Key & Atmospheric Fog

Presets pour créer des atmosphères brumeuses et dramatiques dans Adobe Lightroom Classic et Lightroom CC.

---

## 📦 Presets disponibles

### 1. High-Key Sepia Mist (`High-Key-Sepia-Mist.xmp`)
Style brumeux sépia clair avec tons chauds et doux.
- Exposition haute pour effet éthéré
- Tons beige/crème dans les hautes lumières
- Désaturation douce avec virage sépia chaud
- Vignettage léger et grain film subtil
- **Idéal pour** : portraits, scènes romantiques, paysages brumeux au lever du soleil

### 2. Dark Atmospheric Fog (`Dark-Atmospheric-Fog.xmp`)
Style atmosphérique sombre et froid avec effet brumeux cinématographique.
- Exposition réduite pour ambiance dramatique
- Tons bleu-gris froids
- Vignettage fort et grain cinématographique
- Courbe en S pour contraste prononcé
- **Idéal pour** : paysages urbains, architecture, forêts, scènes nocturnes

---

## 🛠️ Installation

### Lightroom Classic
1. Ouvrez **Lightroom Classic**
2. Allez dans le module **Développement**
3. Dans le panneau **Presets** (gauche), cliquez sur le **+** puis **Importer des presets...**
4. Sélectionnez les fichiers `.xmp` téléchargés
5. Les presets apparaîtront dans le panneau Presets

### Lightroom CC (Cloud)
1. Ouvrez **Adobe Lightroom** (version CC)
2. Dans le module **Édition**, cliquez sur l'icône **Presets** (☰)
3. Cliquez sur les **trois points** (...) puis **Importer des presets**
4. Sélectionnez les fichiers `.xmp`

---

## 🎨 Utilisation

### Appliquer un preset
1. Sélectionnez une photo dans votre bibliothèque
2. Passez en mode **Développement** (Lightroom Classic) ou **Édition** (Lightroom CC)
3. Cliquez sur le preset souhaité dans le panneau Presets
4. Le preset s'applique instantanément

### Ajustements recommandés après application

#### High-Key Sepia Mist
- **Exposition** : ajustez selon la luminosité de votre photo (+0.3 à +1.0)
- **Blancs / Noirs** : affinez pour éviter la surexposition
- **Température** : montez légèrement si l'effet sépia paraît trop froid
- **Saturation** : réduisez davantage pour un rendu plus désaturé

#### Dark Atmospheric Fog
- **Exposition** : baissez si la scène est déjà sombre (-0.2 à -0.6)
- **Noirs** : ajustez selon la profondeur d'ombre souhaitée
- **Vignettage** : diminuez si l'effet est trop prononcé (-20 à -40)
- **Clarté** : augmentez légèrement pour révéler des détails dans la brume

## 💡 Conseils

| Type de photo | Preset recommandé | Conseil |
|---|---|---|
| Portrait en plein air | High-Key Sepia Mist | Fonctionne mieux avec fond neutre ou végétation |
| Paysage brumeux matin | High-Key Sepia Mist | Réduisez le vignettage pour les grands formats |
| Forêt / sous-bois | Dark Atmospheric Fog | Montez légèrement les ombres (+10) |
| Architecture urbaine | Dark Atmospheric Fog | Augmentez la clarté (+10) pour les détails |
| Scène nocturne | Dark Atmospheric Fog | Réduisez l'exposition de -0.2 supplémentaire |
| Portrait dramatique | Dark Atmospheric Fog | Atténuez le vignettage pour le visage |
